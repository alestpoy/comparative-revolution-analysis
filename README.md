# Comparative Revolution Analysis Project

## Purpose

This repository serves as a structured research project comparing the Russian Revolution (1917) and the Chinese Revolution (mid-20th century). The project aims to facilitate clear, comparative analysis of their causes, key ideological influences, and the role of anti-imperialist sentiment.

## Structure

- **comparative-analysis/**: Contains final synthesis and comparative documents.
- **russian-revolution/**: Contains analysis of the Russian Revolution, subdivided into:
  - `causes.md`: Analysis of causes.
  - `ideology.md`: Analysis of ideological influences.
  - `anti-imperialism.md`: Analysis of anti-imperialist sentiment.
- **chinese-revolution/**: Contains analysis of the Chinese Revolution, subdivided similarly.

## Workflow

Research tasks are tracked via GitHub issues. Each issue corresponds to a specific analysis component. Development follows a feature-branch and pull-request workflow to ensure structured collaboration and review.
